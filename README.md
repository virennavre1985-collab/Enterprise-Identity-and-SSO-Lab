# Enterprise-Identity-and-SSO-Lab
This project demonstrates the design and implementation of an enterprise Identity and Access Management (IAM) solution using Microsoft Active Directory, Keycloak, LDAP, Kerberos, and Jenkins. The lab simulates a production-like environment where Active Directory acts as the centralized identity provider, Keycloak provides authentication and SSO and Jenkins is integrated using OpenID Connect (OIDC).
The solution includes LDAP User Federation, Active Directory group synchronization, Role-Based Access Control (RBAC), and Kerberos-based authentication to provide secure, centralized identity management across applications. The environment is deployed using Windows Server, Ubuntu Server, Docker Compose, PostgreSQL, and Jenkins, with a strong focus on enterprise authentication workflows, security best practices, and troubleshooting.
This repository documents the complete implementation, architecture, configuration steps, authentication flow, troubleshooting scenarios, and lessons learned while building the lab.
